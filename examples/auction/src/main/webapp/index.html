auction demonstration

This auction application is a demonstration of an some of the features of Jetty Hightide.
This demo has been implemented using a combination of DWR and cometd:

  • Logon using any username
  • The item catalog is a normal Ajax update mechanism, retrieving data from a database via DWR calls.
  • When bids are made against items, these are published on a JMS topic. The JMS messages are transported with comet style Ajax push, so they update in realtime on other browsers. NB. To test this, bid against yourself in another browser instance or on another computer (due to browser connection limitations, you won't be able to use another window or tab of the same browser).
  • The same JMS mechanism has been used to implement chatrooms per item so you can chat to other parties interested in the item
